# Security Policy

Latest local workflow material reviewed: July 2, 2026.

Do not open a public issue with private education records, course exports,
recording links, credentials, tokens, or source materials.

If a secret or private course artifact is exposed:

1. remove public access to the artifact;
2. rotate any affected credential or link;
3. preserve enough metadata to understand the exposure without copying the
   sensitive content into the public issue;
4. report the issue through a private channel.

This repository publishes the workflow boundary. It is not a place to submit
course materials, transcripts, student records, generated books, or release
bundles.
